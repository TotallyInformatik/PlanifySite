What is Planify?

Planify is a planning software that is designed to be easy to use and beautiful.

It offers:

Tasks

  • Heading
  • Description for Details
  • Set When to Start, When to Stop
  • Separate Sections for Finished and Unfinished
  • Draggable
  • Sorts Automatically via Time

Notes

  • Heading
  • Description for Details
  • Color (Maybe use it as a indicator of urgency?)
  • Large Panel for Multiple Notes
  • Position Whereever you want!

Longterm-Goals

  • Heading
  • Description for Details
  • Unlimited Panels for each Year
  • Panels divided into Phases of the Year
  • Freely Draggable to indicate Deadline

Why Planify?

Intuitive Design

Planify was designed to be as easy to use as possible. Anybody can download Planify and start planning their day without the need to read long instructions etc.

The design is easy to understand, and even if not, tooltips and animations guide the user towards their destination.

Beautiful UI

Beautiful sceneries, excellent fonts and incredibly smooth animations distinguish this Application from any other. Instead of a dull, boring, and uninteresting UI, Planify offers an Interface that is sure to please the user and keep them occupied.

Upcoming Features

The features mentioned above are by far not the only features that will exist. Currently, Planify has only just been released. Many more complex features follow.

Open Source

The code of Planify is purely open source. Anybody can modify the code and change their experience according to their needs.
